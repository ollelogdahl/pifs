# PiFS
Olle Logdahl, 29 August 2019

---
**PiFS** is a quirky file 'compressor' that maps bytes to indexes in Pi.


## Installation / Getting Started
- [Download Lastest Build](https://github.com/ollelogdahl/pifs/releases/)
